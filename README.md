# 💼 Calculamiga — Calculadora da Trabalhadora Horista

Projeto desenvolvido em Python para cálculo de salário de trabalhadores horistas, considerando horas regulares, horas extras, adicionais e descontos.

## 📌 Sobre o projeto

A Calculamiga foi criada como atividade acadêmica do curso de Engenharia de Software da UMC com o objetivo de aplicar conceitos fundamentais de lógica de programação, processamento de dados e automação de cálculos salariais.

O sistema recebe informações do trabalhador e gera automaticamente um demonstrativo salarial completo diretamente no terminal.

---

## 🚀 Funcionalidades

- Cálculo de salário regular
- Cálculo de horas extras
- Aplicação de adicional percentual
- Cálculo de salário bruto e líquido
- Aplicação de descontos
- Recibo formatado no terminal
- Interface em ASCII Art

---

## 🛠️ Tecnologias utilizadas

- Python 3
- Lógica de Programação
- Manipulação de Strings
- Operações Matemáticas

---

## 📷 Exemplo de execução

```bash
Insira o seu nome: Eduarda
Insira o seu CPF: 000.000.000-00
Insira o seu cargo: Desenvolvedora
Insira o total de horas regulares trabalhadas: 160
Insira o valor da hora trabalhada: 40.95
```

Resultado:

```bash
RECIBO DE PAGAMENTO

O salário bruto tem valor igual a: R$ XXXX.XX
O salário líquido tem valor igual a: R$ XXXX.XX
```
